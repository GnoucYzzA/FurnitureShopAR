# furniturearshop

## Project setup
```
yarn install
```
## Install vuetify
```
vue add vuetify
```
## Install material design icon
```
yarn add material-design-icons-iconfont -D
```
## Install fontawesome
```
yarn add @fortawesome/fontawesome-free -D
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
